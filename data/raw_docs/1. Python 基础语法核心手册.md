# Python 基础语法核心手册

## 一、变量与数据类型

### 1.1 变量命名规则

- 只能由字母、数字、下划线组成，不能以数字开头
- 区分大小写
- 不能使用 Python 关键字（如 if、for、class 等）

```python
# 变量赋值示例
name = "Python学习"
age = 20
is_student = True
```

### 1.2 核心数据类型

| 类型  | 说明   | 示例                           |
| ----- | ------ | ------------------------------ |
| int   | 整数   | `a = 100`                      |
| float | 浮点数 | `b = 3.14`                     |
| str   | 字符串 | `c = "hello world"`            |
| bool  | 布尔值 | `d = True / False`             |
| list  | 列表   | `e = [1,2,3,"test"]`           |
| tuple | 元组   | `f = (1,2,3)`                  |
| dict  | 字典   | `g = {"name":"Tom", "age":18}` |
| set   | 集合   | `h = {1,2,3}`                  |

```python
# 数字类型
int_num = 100  # 整型
float_num = 3.14  # 浮点型
complex_num = 1+2j  # 复数

# 序列类型
str_data = "hello python"  # 字符串
list_data = [1, 2, 3, "test"]  # 列表
tuple_data = (1, 2, 3)  # 元组

# 映射类型
dict_data = {"name": "Tom", "age": 20}  # 字典

# 集合类型
set_data = {1, 2, 3, 3}  # 集合，自动去重

# 布尔类型
bool_true = True
bool_false = False
```

### 1.3 数据类型转换

```python
# 常用类型转换
int("123")  # 字符串转整数
float("3.14")  # 字符串转浮点数
str(100)  # 数字转字符串
list((1,2,3))  # 元组转列表
tuple([1,2,3])  # 列表转元组
```

## 三、流程控制

### 3.1 条件语句

```python
# if-elif-else 结构
score = 85
if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

### 3.2. 循环语句

#### for 循环

```python
# 遍历列表
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

# 范围循环
for i in range(1, 10):  # 1到9
    print(i)
```

**While 循环**

```python
count = 0
while count < 5:
    print(count)
    count += 1
```

#### 循环控制

- `break`：终止整个循环
- `continue`：跳过当前循环，进入下一轮
- `pass`：空语句，占位用

## 四、函数

### 4.1. 函数定义与调用

```python
# 无返回值函数
def say_hello(name):
    print(f"Hello, {name}!")

say_hello("Python")

# 有返回值函数
def add(a, b):
    return a + b

result = add(3, 5)
print(result)
```

### 4.2. 函数参数

- 位置参数：按顺序传递
- 默认参数：设置默认值，不传则使用默认
- 关键字参数：指定参数名传递，顺序无关
- 不定长参数：`*args` 接收元组，`**kwargs` 接收字典

```python
def func(a, b=10, *args, **kwargs):
    print(a, b)
    print(args)
    print(kwargs)

func(1, 2, 3, 4, name="Tom", age=18)
```

### 4.3 lambda 匿名函数

```python
# 语法：lambda 参数: 表达式
add = lambda x, y: x + y
print(add(2, 3))

# 常用于排序
nums = [(1, 3), (4, 1), (2, 5)]
nums.sort(key=lambda x: x[1])
print(nums)
```

## 五、面向对象编程

### 5.1. 类与对象定义

```python
class Student:
    # 构造方法
    def __init__(self, name, age):
        self.name = name  # 实例属性
        self.age = age

    # 实例方法
    def introduce(self):
        print(f"我叫{self.name}，今年{self.age}岁")

# 创建对象
stu = Student("Tom", 18)
stu.introduce()
```

### 5.2. 三大特性

#### 封装

通过访问控制实现，Python 中以单下划线`_`表示受保护，双下划线`__`表示私有

```python
class Person:
    def __init__(self, name):
        self.__name = name  # 私有属性

    def get_name(self):
        return self.__name

p = Person("Jack")
print(p.get_name())
```

**继承**

```python
# 父类
class Animal:
    def eat(self):
        print("吃东西")

# 子类继承父类
class Dog(Animal):
    def bark(self):
        print("汪汪叫")

dog = Dog()
dog.eat()  # 继承父类方法
dog.bark()
```

**多态**

```python
class Cat(Animal):
    def eat(self):
        print("猫吃猫粮")

class Dog(Animal):
    def eat(self):
        print("狗吃狗粮")

def feed_animal(animal):
    animal.eat()

feed_animal(Cat())
feed_animal(Dog())
```
