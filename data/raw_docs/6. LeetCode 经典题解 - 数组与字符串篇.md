# LeetCode 经典题解 - 数组与字符串篇

## 题1：两数之和（LeetCode 1，简单）

### 题目描述

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。 你可以假设每种输入只会对应

### 解题思路

- 暴力解法：双重循环遍历，时间复杂度O(n²)，空间复杂度O(1)，效率低 
- 最优解法：哈希表法，遍历数组时，用哈希表存储已遍历元素的值和下标，每次检查`target - 当前值`是否在哈希表中，存在则直接返回结果 
- 时间复杂度O(n)，空间复杂度O(n)，一次遍历即可完成

### 代码实现

```python
# Python实现
def twoSum(nums, target):
    hash_map = {}
    for index, num in enumerate(nums):
        complement = target - num
        if complement in hash_map:
            return [hash_map[complement], index]
        hash_map[num] = index
    return []
```

```java
// Java实现
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            int complement = target - nums[i];
            if (map.containsKey(complement)) {
                return new int[]{map.get(complement), i};
            }
            map.put(nums[i], i);
        }
        return new int[0];
    }
}
```

### 复杂度分析

- 时间复杂度：O (n)，仅遍历数组一次，哈希表的查找和插入都是 O (1)
- 空间复杂度：O (n)，最多存储 n 个元素到哈希表中

## 题 2：最长回文子串（LeetCode 5，中等）

### 题目描述

给你一个字符串 s，找到 s 中最长的回文子串。

回文串是指正着读和反着读都一样的字符串。

### 解题思路

- 中心扩展法：回文串的核心是对称，枚举每一个可能的回文中心，向左右两边扩展，直到字符不相等，记录最大长度
- 回文中心分为两种：奇数长度（中心为单个字符）、偶数长度（中心为两个相邻字符）
- 时间复杂度 O (n²)，空间复杂度 O (1)，实现简单，效率优于暴力解法

### 代码实现

```python
# Python实现
class Solution:
    def longestPalindrome(self, s: str) -> str:
        def expand(left, right):
            # 向左右扩展，返回最长回文串的边界
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return left + 1, right - 1
        
        start = 0
        end = 0
        # 枚举所有回文中心
        for i in range(len(s)):
            # 奇数长度回文
            l1, r1 = expand(i, i)
            # 偶数长度回文
            l2, r2 = expand(i, i+1)
            # 取最长的
            if r1 - l1 > end - start:
                start, end = l1, r1
            if r2 - l2 > end - start:
                start, end = l2, r2
        return s[start:end+1]
```

```java
// Java实现
class Solution {
    public String longestPalindrome(String s) {
        if (s == null || s.length() < 1) return "";
        int start = 0, end = 0;
        for (int i = 0; i < s.length(); i++) {
            int len1 = expand(s, i, i);
            int len2 = expand(s, i, i+1);
            int len = Math.max(len1, len2);
            if (len > end - start) {
                start = i - (len - 1) / 2;
                end = i + len / 2;
            }
        }
        return s.substring(start, end + 1);
    }

    private int expand(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        return right - left - 1;
    }
}
```

### 复杂度分析

- 时间复杂度：O (n²)，枚举 n 个中心，每个中心最多扩展 n 次
- 空间复杂度：O (1)，仅使用常数级额外空间

## 题 3：反转字符串（LeetCode 344，简单）

### 题目描述

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 char [] 的形式给出。

不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用 O (1) 的额外空间解决这一问题。

### 解题思路

- 双指针法：左指针指向数组头部，右指针指向数组尾部，交换两个指针指向的元素，然后左指针右移，右指针左移，直到两指针相遇
- 时间复杂度 O (n)，空间复杂度 O (1)，原地反转，最优解法

### 代码实现

```python
# Python实现
class Solution:
    def reverseString(self, s: List[str]) -> None:
        left, right = 0, len(s) - 1
        while left < right:
            s[left], s[right] = s[right], s[left]
            left += 1
            right -= 1
```

```java
# Python实现
class Solution:
    def reverseString(self, s: List[str]) -> None:

```

### 复杂度分析

- 时间复杂度：O (n)，共交换 n/2 次
- 空间复杂度：O (1)，仅使用常数级额外空间
