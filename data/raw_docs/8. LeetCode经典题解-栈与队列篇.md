# LeetCode经典题解-栈与队列篇

## 题1：有效的括号（LeetCode 20，简单） 

### 题目描述 

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。 

有效字符串需满足： 

1. 左括号必须用相同类型的右括号闭合。 
2. 左括号必须以正确的顺序闭合。 
3. 每个右括号都有一个对应的相同类型的左括号。 

### 解题思路 

* 栈解法：遇到左括号入栈，遇到右括号时，检查栈顶是否是对应的左括号，若是则出栈，否则直接返回false - 遍历结束后，栈必须为空，说明所有左括号都已闭合 - 用哈希表存储右括号到左括号的映射，方便快速匹配 - 时间复杂度O(n)，空间复杂度O(n)，最优解法 

### 代码实现

```python
# Python实现
class Solution:
    def isValid(self, s: str) -> bool:
        # 右括号到左括号的映射
        bracket_map = {')':'(', '}':'{', ']':'['}
        stack = []
        for c in s:
            # 右括号
            if c in bracket_map:
                # 栈为空或栈顶不匹配
                if not stack or stack[-1] != bracket_map[c]:
                    return False
                stack.pop()
            # 左括号
            else:
                stack.append(c)
        # 栈必须为空
        return not stack
```

```java
// Java实现
class Solution {
    public boolean isValid(String s) {
        Map<Character, Character> map = new HashMap<>();
        map.put(')', '(');
        map.put('}', '{');
        map.put(']', '[');
        Deque<Character> stack = new LinkedList<>();
        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            if (map.containsKey(c)) {
                if (stack.isEmpty() || stack.peek() != map.get(c)) {
                    return false;
                }
                stack.pop();
            } else {
                stack.push(c);
            }
        }
        return stack.isEmpty();
    }
}
```

### 复杂度分析

- 时间复杂度：O (n)，遍历字符串一次
- 空间复杂度：O (n)，最坏情况全是左括号，栈存储 n 个元素

## 题 2：用栈实现队列（LeetCode 232，简单）

### 题目描述

请你仅使用两个栈实现先入先出队列。队列应当支持一般队列支持的所有操作（push、pop、peek、empty）：

实现 MyQueue 类：

- void push (int x) 将元素 x 推到队列的末尾
- int pop () 从队列的开头移除并返回元素
- int peek () 返回队列开头的元素
- boolean empty () 如果队列为空，返回 true ；否则，返回 false

### 解题思路

- 双栈法：用两个栈，inStack 负责入队操作，outStack 负责出队操作
- 入队：直接 push 到 inStack
- 出队：若 outStack 为空，将 inStack 所有元素依次弹出并 push 到 outStack，此时 outStack 的栈顶就是队头元素，弹出即可
- 时间复杂度：每个元素最多入栈和出栈各 2 次，均摊时间复杂度 O (1)
- 空间复杂度 O (n)，存储 n 个元素

### 代码实现

```python
# Python实现
class MyQueue:

    def __init__(self):
        self.in_stack = []
        self.out_stack = []

    def push(self, x: int) -> None:
        self.in_stack.append(x)

    def pop(self) -> int:
        self._transfer()
        return self.out_stack.pop()

    def peek(self) -> int:
        self._transfer()
        return self.out_stack[-1]

    def empty(self) -> bool:
        return not self.in_stack and not self.out_stack
    
    # 核心：in_stack元素转移到out_stack
    def _transfer(self):
        if not self.out_stack:
            while self.in_stack:
                self.out_stack.append(self.in_stack.pop())
```

```java
// Java实现
class MyQueue {
    private Deque<Integer> inStack;
    private Deque<Integer> outStack;

    public MyQueue() {
        inStack = new LinkedList<>();
        outStack = new LinkedList<>();
    }
    
    public void push(int x) {
        inStack.push(x);
    }
    
    public int pop() {
        transfer();
        return outStack.pop();
    }
    
    public int peek() {
        transfer();
        return outStack.peek();
    }
    
    public boolean empty() {
        return inStack.isEmpty() && outStack.isEmpty();
    }

    private void transfer() {
        if (outStack.isEmpty()) {
            while (!inStack.isEmpty()) {
                outStack.push(inStack.pop());
            }
        }
    }
}
```

### 复杂度分析

- 时间复杂度：均摊 O (1)，每个元素仅会被转移一次
- 空间复杂度：O (n)，存储队列的所有元素

## 题 3：最小栈（LeetCode 155，中等）

### 题目描述

设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

- MinStack () 初始化堆栈对象。
- void push (int val) 将元素 val 推入堆栈。
- void pop () 删除堆栈顶部的元素。
- int top () 获取堆栈顶部的元素。
- int getMin () 获取堆栈中的最小元素。

### 解题思路

- 辅助栈法：使用两个栈，主栈存储所有元素，辅助栈存储每个元素入栈时的当前最小值
- 入栈：主栈 push 元素，辅助栈 push 当前最小值（min (辅助栈栈顶，新元素)）
- 出栈：主栈和辅助栈同时 pop
- 取最小值：直接取辅助栈栈顶元素
- 所有操作时间复杂度均为 O (1)，空间复杂度 O (n)

### 代码实现

```python
# Python实现
class MinStack:

    def __init__(self):
        self.main_stack = []
        self.min_stack = []

    def push(self, val: int) -> None:
        self.main_stack.append(val)
        # 辅助栈push当前最小值
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))

    def pop(self) -> None:
        self.main_stack.pop()
        self.min_stack.pop()

    def top(self) -> int:
        return self.main_stack[-1]

    def getMin(self) -> int:
        return self.min_stack[-1]
```

```java
// Java实现
class MinStack {
    private Deque<Integer> mainStack;
    private Deque<Integer> minStack;

    public MinStack() {
        mainStack = new LinkedList<>();
        minStack = new LinkedList<>();
    }
    
    public void push(int val) {
        mainStack.push(val);
        if (minStack.isEmpty()) {
            minStack.push(val);
        } else {
            minStack.push(Math.min(val, minStack.peek()));
        }
    }
    
    public void pop() {
        mainStack.pop();
        minStack.pop();
    }
    
    public int top() {
        return mainStack.peek();
    }
    
    public int getMin() {
        return minStack.peek();
    }
}
```

### 复杂度分析

- 时间复杂度：所有操作均为 O (1)
- 空间复杂度：O (n)，最坏情况存储 n 个元素的最小值
