# Java基础语法核心手册

## 一、基础概念与环境

### 1. 程序入口

Java程序的唯一入口是`main`方法，固定格式：

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello Java!");
    }
}
```

* 文件名必须与`public`类名完全一致，后缀为`.java`

* 编译：`javac HelloWorld.java` 生成`.class`字节码文件

* 运行：`java HelloWorld`

## 二、变量与数据类型

### 1. 变量定义

Java 是强类型语言，变量必须先声明类型再使用，命名规则与 Python 一致，驼峰命名法规范。

```java
// 数据类型 变量名 = 初始值;
String name = "Java学习";
int age = 20;
boolean isStudent = true;
```

### 2. 八大基本数据类型

| 类型    | 说明         | 字节数 | 取值范围          |
| ------- | ------------ | ------ | ----------------- |
| byte    | 字节型       | 1      | -128 ~ 127        |
| short   | 短整型       | 2      | -32768 ~ 32767    |
| int     | 整型         | 4      | -2^31 ~ 2^31-1    |
| long    | 长整型       | 8      | -2^63 ~ 2^63-1    |
| float   | 单精度浮点型 | 4      | 有效数字 6-7 位   |
| double  | 双精度浮点型 | 8      | 有效数字 15 位    |
| char    | 字符型       | 2      | 单个 Unicode 字符 |
| boolean | 布尔型       | 1      | true/false        |

### 3. 引用数据类型

- 类（Class）、接口（Interface）、数组（Array）、字符串（String）
- 引用类型默认值为`null`

```java
// 数组定义
int[] nums = new int[5]; // 长度为5的int数组
int[] arr = {1,2,3,4,5}; // 静态初始化

// 字符串
String str = "hello java";
```

### 4. 类型转换

- 自动类型转换：小范围 → 大范围（无风险）
- 强制类型转换：大范围 → 小范围（有精度丢失风险）

```java
// 自动转换
int a = 10;
double b = a;

// 强制转换
double c = 3.14;
int d = (int) c; // d=3
```

## 三、流程控制语句

### 1. 条件语句

```java
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}

// switch语句（支持int、String、枚举）
int day = 1;
switch (day) {
    case 1:
        System.out.println("周一");
        break;
    case 2:
        System.out.println("周二");
        break;
    default:
        System.out.println("其他");
        break;
}
```

### 2. 循环语句

```java
// for循环
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

// 增强for循环（遍历数组/集合）
int[] arr = {1,2,3,4,5};
for (int num : arr) {
    System.out.println(num);
}

// while循环
int count = 0;
while (count < 5) {
    System.out.println(count);
    count++;
}

// do-while循环（至少执行一次）
int num = 0;
do {
    System.out.println(num);
    num++;
} while (num < 5);
```

### 循环控制

- `break`：终止循环
- `continue`：跳过当前循环，进入下一轮
- `return`：结束方法

## 四、面向对象编程

### 1. 类与对象

```java
// 类定义
public class Student {
    // 成员属性
    private String name;
    private int age;

    // 构造方法
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 成员方法
    public void introduce() {
        System.out.println("我叫" + name + "，今年" + age + "岁");
    }

    // getter/setter
    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}

// 创建对象
public class Test {
    public static void main(String[] args) {
        Student stu = new Student("Tom", 18);
        stu.introduce();
        stu.setName("Jack");
        System.out.println(stu.getName());
    }
}
```

### 2. 三大特性

#### 封装

通过访问权限修饰符实现：

- `private`：仅本类可访问
- `default`：本类 + 同包可访问
- `protected`：本类 + 同包 + 子类可访问
- `public`：所有类可访问

#### 继承

使用`extends`关键字，Java 单继承，多实现

```java
// 父类
public class Animal {
    public void eat() {
        System.out.println("吃东西");
    }
}

// 子类
public class Dog extends Animal {
    public void bark() {
        System.out.println("汪汪叫");
    }

    // 方法重写
    @Override
    public void eat() {
        System.out.println("狗吃狗粮");
    }
}
```

#### 多态

- 前提：继承、方法重写、父类引用指向子类对象

```java
public class Cat extends Animal {
    @Override
    public void eat() {
        System.out.println("猫吃猫粮");
    }
}

public class Test {
    public static void main(String[] args) {
        Animal animal1 = new Dog();
        Animal animal2 = new Cat();
        animal1.eat();
        animal2.eat();
    }
}
```

## 五、异常处理

### 核心异常处理结构

```java
public class ExceptionTest {
    public static void main(String[] args) {
        try {
            // 可能出现异常的代码
            int a = 10 / 0;
        } catch (ArithmeticException e) {
            // 捕获并处理异常
            System.out.println("算术异常：" + e.getMessage());
            e.printStackTrace();
        } catch (Exception e) {
            // 捕获其他异常
            System.out.println("其他异常");
        } finally {
            // 无论是否异常，都会执行的代码
            System.out.println("最终执行");
        }
    }
}
```

### 常见异常类型

- 运行时异常：`NullPointerException`、`ArrayIndexOutOfBoundsException`、`ClassCastException`等
- 受检异常：`IOException`、`SQLException`等，必须显式处理

## 六、集合框架

### 常用集合

```java
import java.util.ArrayList;
import java.util.HashMap;
import java.util.HashSet;
import java.util.List;
import java.util.Map;
import java.util.Set;

public class CollectionTest {
    public static void main(String[] args) {
        // List 有序、可重复
        List<String> list = new ArrayList<>();
        list.add("a");
        list.add("b");
        System.out.println(list.get(0));

        // Set 无序、不可重复
        Set<Integer> set = new HashSet<>();
        set.add(1);
        set.add(2);

        // Map 键值对，key不可重复
        Map<String, Integer> map = new HashMap<>();
        map.put("Tom", 18);
        map.put("Jack", 20);
        System.out.println(map.get("Tom"));
    }
}
```
