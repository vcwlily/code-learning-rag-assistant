# LeetCode经典题解-动态规划入门篇

## 题1：爬楼梯（LeetCode 70，简单） 

### 题目描述 

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。 

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？ 

### 解题思路 

- 动态规划核心拆解：  
  1. **状态定义**：`dp[i]` 表示爬到第i阶楼梯的总方法数  
  2. **状态转移方程**：`dp[i] = dp[i-1] + dp[i-2]`     
     * 爬到第i阶，只有两种路径：从i-1阶爬1步上来，或从i-2阶爬2步上来，总方法数为两者之和  
  3. **初始边界**：`dp[1] = 1`（1阶只有1种方法），`dp[2] = 2`（2阶有2种方法：1+1 / 直接2步） 
- 空间优化：无需存储整个dp数组，仅用两个变量保存前两个状态即可，空间复杂度从O(n)优化到O(1) 
- 本质是斐波那契数列的应用场景，是动态规划最基础的入门题 

### 代码实现

```python
# Python实现1：基础动态规划（完整dp数组）
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        dp = [0] * (n + 1)
        dp[1] = 1
        dp[2] = 2
        for i in range(3, n + 1):
            dp[i] = dp[i-1] + dp[i-2]
        return dp[n]

# Python实现2：空间优化版（推荐）
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        # 前两个状态：a=dp[i-2], b=dp[i-1]
        a, b = 1, 2
        for _ in range(3, n + 1):
            a, b = b, a + b
        return b
```

```java
    public int climbStairs(int n) {
        if (n <= 2) return n;
        int a = 1, b = 2;
        for (int i = 3; i <= n; i++) {
            int temp = a + b;
            a = b;
            b = temp;
        }
        return b;
    }
}
```

### 复杂度分析

- 基础版：时间复杂度 O (n)，空间复杂度 O (n)
- 优化版：时间复杂度 O (n)，空间复杂度 O (1)

## 题 2：使用最小花费爬楼梯（LeetCode 746，简单）

### 题目描述

给你一个整数数组 cost ，其中 cost [i] 是从楼梯第 i 个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。

你可以选择从下标为 0 或下标为 1 的台阶开始爬楼梯。

请你计算并返回达到楼梯顶部的最低花费。

### 解题思路

- 动态规划核心拆解：
  1. **状态定义**：`dp[i]` 表示到达第 i 个台阶所需的最小花费
  2. 状态转移方程：`dp[i] = min(dp[i-1], dp[i-2]) + cost[i]`
     - 到达第 i 阶，只能从 i-1 阶或 i-2 阶上来，取两者的最小花费，加上当前台阶的费用
  3. **初始边界**：`dp[0] = cost[0]`，`dp[1] = cost[1]`（可以从 0 或 1 阶起步）
  4. 最终结果：楼顶在最后一个台阶之后，所以返回`min(dp[n-1], dp[n-2])`（n 为 cost 数组长度）
- 同样可做空间优化，仅用两个变量保存前两个状态，无需完整 dp 数组

### 代码实现

```python
# Python实现1：基础动态规划
class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        n = len(cost)
        if n <= 1:
            return 0
        dp = [0] * n
        dp[0] = cost[0]
        dp[1] = cost[1]
        for i in range(2, n):
            dp[i] = min(dp[i-1], dp[i-2]) + cost[i]
        # 顶部在最后一阶之后，取最后两个的最小值
        return min(dp[-1], dp[-2])

# Python实现2：空间优化版
class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        n = len(cost)
        if n <= 1:
            return 0
        a, b = cost[0], cost[1]
        for i in range(2, n):
            a, b = b, min(a, b) + cost[i]
        return min(a, b)
```

```java

            a = b;
            b = temp;
        }
        return Math.min(a, b);
    }
}
```

### 复杂度分析

- 基础版：时间复杂度 O (n)，空间复杂度 O (n)
- 优化版：时间复杂度 O (n)，空间复杂度 O (1)

## 题 3：打家劫舍（LeetCode 198，中等）

### 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个代表每个房屋存放金额的非负整数数组，计算你 不触动警报装置的情况下 ，一夜之内能够偷窃到的最高金额。

### 解题思路

- 动态规划核心拆解：
  1. **状态定义**：`dp[i]` 表示前 i 间房屋能偷窃到的最高金额
  2. 状态转移方程：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`
     - 两种选择：① 不偷第 i 间房，最高金额等于 dp [i-1]；② 偷第 i 间房，就不能偷 i-1 间，最高金额等于 dp [i-2] + 当前房屋金额，取两者最大值
  3. 初始边界
     - 只有 1 间房：`dp[0] = nums[0]`
     - 有 2 间房：`dp[1] = max(nums[0], nums[1])`
- 空间优化：仅用两个变量保存前两个状态，空间复杂度优化到 O (1)

### 代码实现

```python
# Python实现1：基础动态规划
class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return nums[0]
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        for i in range(2, n):
            dp[i] = max(dp[i-1], dp[i-2] + nums[i])
        return dp[-1]

# Python实现2：空间优化版（推荐）
class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return nums[0]
        # a=dp[i-2], b=dp[i-1]
        a, b = nums[0], max(nums[0], nums[1])
        for i in range(2, n):
            a, b = b, max(b, a + nums[i])
        return b
```

### 复杂度分析

- 基础版：时间复杂度 O (n)，空间复杂度 O (n)
- 优化版：时间复杂度 O (n)，空间复杂度 O (1)

## 题 4：最大子数组和（LeetCode 53，中等）

### 题目描述

给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组 是数组中的一个连续部分。

### 解题思路

- 动态规划（Kadane 算法）核心拆解：
  1. **状态定义**：`dp[i]` 表示以第 i 个元素结尾的连续子数组的最大和
  2. 状态转移方程：`dp[i] = max(nums[i], dp[i-1] + nums[i])`
     - 两种选择：① 以当前元素为起点，重新开始子数组；② 接在之前的子数组后面，取两者最大值
  3. **初始边界**：`dp[0] = nums[0]`
  4. 最终结果：遍历 dp 数组，取最大值，即为整个数组的最大子数组和
- 空间优化：无需 dp 数组，仅用一个变量保存前一个状态，另一个变量保存全局最大值，空间复杂度 O (1)

### 代码实现

```python
# Python实现1：基础动态规划
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        dp = [0] * n
        dp[0] = nums[0]
        for i in range(1, n):
            dp[i] = max(nums[i], dp[i-1] + nums[i])
        return max(dp)

# Python实现2：空间优化版（推荐）
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        pre = nums[0]  # 保存dp[i-1]
        max_sum = pre  # 保存全局最大值
        for i in range(1, len(nums)):
            pre = max(nums[i], pre + nums[i])
            max_sum = max(max_sum, pre)
        return max_sum
```

```java
// Java实现1：基础动态规划
class Solution {
    public int maxSubArray(int[] nums) {
        int n = nums.length;
        int[] dp = new int[n];
        dp[0] = nums[0];
        for (int i = 1; i < n; i++) {
            dp[i] = Math.max(nums[i], dp[i-1] + nums[i]);
        }
        // 找dp数组的最大值
        int maxSum = dp[0];
        for (int num : dp) {
            maxSum = Math.max(maxSum, num);
        }
        return maxSum;
    }
}

// Java实现2：空间优化版
class Solution {
    public int maxSubArray(int[] nums) {
        int pre = nums[0];
        int maxSum = pre;
        for (int i = 1; i < nums.length; i++) {
            pre = Math.max(nums[i], pre + nums[i]);
            maxSum = Math.max(maxSum, pre);
        }
        return maxSum;
    }
}
```

### 复杂度分析

- 基础版：时间复杂度 O (n)，空间复杂度 O (n)
- 优化版：时间复杂度 O (n)，空间复杂度 O (1)

## 题 5：零钱兑换（LeetCode 322，中等）

### 题目描述

给你一个整数数组 coins ，表示不同面额的硬币；以及一个整数 amount ，表示总金额。

计算并返回可以凑成总金额所需的 最少的硬币个数 。如果没有任何一种硬币组合能组成总金额，返回 -1 。

你可以认为每种硬币的数量是无限的。

### 解题思路

- 完全背包问题的经典入门题，动态规划核心拆解：
  1. **状态定义**：`dp[i]` 表示凑成金额 i 所需的最少硬币个数
  2. 状态转移方程：`dp[i] = min(dp[i], dp[i - coin] + 1)`
     - 遍历每个硬币，若当前硬币面额 coin ≤ i，说明可以使用该硬币，凑成 i 的最少硬币数 = 凑成 i-coin 的最少硬币数 + 1（当前这枚硬币），取所有可能中的最小值
  3. 初始边界：
     - `dp[0] = 0`（凑成金额 0，需要 0 个硬币）
     - 其余 dp 值初始化为一个大于 amount 的数（如 amount+1），表示初始状态无法凑成，方便后续取最小值
  4. 最终结果：若`dp[amount]`仍大于 amount，说明无法凑成，返回 - 1，否则返回`dp[amount]`

### 代码实现

```python
# Python实现
class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        # 初始化dp数组，最大值设为amount+1（最多需要amount个1元硬币）
        dp = [amount + 1] * (amount + 1)
        dp[0] = 0
        # 遍历每个金额
        for i in range(1, amount + 1):
            # 遍历每个硬币
            for coin in coins:
                if coin <= i:
                    dp[i] = min(dp[i], dp[i - coin] + 1)
        # 判断是否能凑成
        return dp[amount] if dp[amount] <= amount else -1
```

```java
// Java实现
class Solution {
    public int coinChange(int[] coins, int amount) {
        int max = amount + 1;
        int[] dp = new int[amount + 1];
        // 初始化数组
        Arrays.fill(dp, max);
        dp[0] = 0;
        for (int i = 1; i <= amount; i++) {
            for (int coin : coins) {
                if (coin <= i) {
                    dp[i] = Math.min(dp[i], dp[i - coin] + 1);
                }
            }
        }
        return dp[amount] > amount ? -1 : dp[amount];
    }
}
```

### 复杂度分析

- 时间复杂度：O (amount * k)，其中 k 是硬币的种类数，需要遍历每个金额和每个硬币
- 空间复杂度：O (amount)，dp 数组的长度为 amount+1

## 题 6：最长递增子序列（LeetCode 300，中等）

### 题目描述

给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。

子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列。

### 解题思路

- 子序列动态规划入门经典题，核心拆解：

  1. **状态定义**：`dp[i]` 表示以第 i 个元素结尾的最长严格递增子序列的长度

  2. 状态转移方程：`dp[i] = max(dp[i], dp[j] + 1)`

     （其中 0 ≤ j <i，且 nums [j] < nums [i]）

     - 遍历 i 之前的所有元素 j，若 nums [j] < nums [i]，说明可以把 nums [i] 接在以 j 结尾的递增子序列后面，长度 + 1，取所有可能中的最大值

  3. **初始边界**：dp 数组所有元素初始化为 1，因为每个元素自身就是一个长度为 1 的子序列

  4. 最终结果：dp 数组中的最大值，即为整个数组的最长递增子序列长度

### 代码实现

```python
# Python实现
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        dp = [1] * n
        for i in range(1, n):
            for j in range(i):
                if nums[j] < nums[i]:
                    dp[i] = max(dp[i], dp[j] + 1)
        return max(dp)
```

```java
// Java实现
class Solution {
    public int lengthOfLIS(int[] nums) {
        int n = nums.length;
        if (n == 0) return 0;
        int[] dp = new int[n];
        Arrays.fill(dp, 1);
        int maxLen = 1;
        for (int i = 1; i < n; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[j] < nums[i]) {
                    dp[i] = Math.max(dp[i], dp[j] + 1);
                }
            }
            maxLen = Math.max(maxLen, dp[i]);
        }
        return maxLen;
    }
}
```

### 复杂度分析

- 时间复杂度：O (n²)，两层循环遍历数组
- 空间复杂度：O (n)，dp 数组的长度为 n
