# Python常见编程报错及解决方案汇总

## 一、语法错误类（SyntaxError）

### 1. SyntaxError: invalid syntax 

**报错原因**：代码语法格式错误，新手最常见报错 

**常见场景**：

- 括号、引号不配对 
- 冒号`:`遗漏（if/for/def/class等语句后） 
- -缩进错误 
- 关键字错误使用 

**解决方案**：

```python
# 错误示例1：遗漏冒号
if a > 10
    print("大于10")

# 正确示例
if a > 10:
    print("大于10")

# 错误示例2：引号不配对
print("hello world)

# 正确示例
print("hello world")
```

### 2. IndentationError: expected an indented block

**报错原因**：缩进错误，Python 强制缩进规范，通常 4 个空格

**常见场景**：

- 代码块未缩进
- 缩进空格数不一致
- 混用 Tab 和空格

**解决方案：**

```python
# 错误示例
def add(a, b):
return a + b

# 正确示例
def add(a, b):
    return a + b
```

## 二、名称与类型错误类

### 1. NameError: name 'xxx' is not defined

**报错原因**：使用了未定义的变量 / 函数 / 类

**常见场景**：

- 变量未赋值就使用
- 拼写错误
- 作用域错误（局部变量在全局使用）

**解决方案：**

```python
# 错误示例
print(num)
num = 10

# 正确示例
num = 10
print(num)
```

### 2. TypeError: unsupported operand type(s) for +: 'int' and 'str'

**报错原因**：数据类型不匹配，操作符 / 函数不支持该类型

**常见场景**：

- 数字和字符串直接拼接
- 函数传参类型错误
- 调用非可调用对象（如把变量当函数调用）

**解决方案：**

```python
# 错误示例
age = 18
print("我今年" + age + "岁")

# 正确示例
age = 18
print("我今年" + str(age) + "岁")
print(f"我今年{age}岁")
```

## 三、索引与键值错误类

### 1. IndexError: list index out of range

**报错原因**：序列（列表 / 元组 / 字符串）索引超出范围

**常见场景**：

- 索引超过列表最大下标（下标从 0 开始）
- 空列表取值
- 循环中索引越界

**解决方案：**

```python
# 错误示例
nums = [1,2,3]
print(nums[3])  # 最大索引为2

# 正确示例
nums = [1,2,3]
if len(nums) > 3:
    print(nums[3])
```

### 2. KeyError: 'xxx'

**报错原因**：字典中不存在对应的 key

**常见场景**：

- 访问不存在的 key
- key 拼写错误

**解决方案：**

```python
# 错误示例
user = {"name":"Tom", "age":18}
print(user["gender"])

# 正确示例1：先判断key是否存在
if "gender" in user:
    print(user["gender"])

# 正确示例2：使用get方法，设置默认值
print(user.get("gender", "未知"))
```

## 四、属性与导入错误类

### 1. AttributeError: 'xxx' object has no attribute 'yyy'

**报错原因**：对象没有对应的属性 / 方法

**常见场景**：

- 方法 / 属性名拼写错误
- 变量类型与预期不符
- 空对象调用方法

**解决方案：**

```python
# 错误示例
num = 123
num.append(4)  # int类型没有append方法

# 正确示例
nums = [1,2,3]
nums.append(4)
```

### 2. ImportError: No module named 'xxx'

**报错原因**：导入模块失败

**常见场景**：

- 模块未安装
- 模块名拼写错误
- 导入路径错误
- 循环导入

**解决方案：**

1. 检查模块是否安装：`pip install xxx`
2. 核对模块名拼写
3. 检查项目路径与导入语句是否匹配

```python
# 错误示例
import pandans  # 拼写错误

# 正确示例
import pandas
```

## 五、文件与 IO 错误类

### 1. FileNotFoundError: [Errno 2] No such file or directory: 'xxx.txt'

**报错原因**：文件 / 目录不存在

**常见场景**：

- 文件路径错误
- 文件未创建
- 相对路径与工作目录不符

**解决方案：**

1. 核对文件路径是否正确
2. 使用绝对路径
3. 先判断文件是否存在再操作

```python
import os

file_path = "test.txt"
if os.path.exists(file_path):
    with open(file_path, "r", encoding="utf-8") as f:
        print(f.read())
else:
    print("文件不存在")
```

### 2. UnicodeDecodeError: 'gbk' codec can't decode byte

**报错原因**：文件编码不匹配

**常见场景**：

- 读取 utf-8 编码的文件时，默认使用 gbk 编码
- 文件编码与指定编码不符

解决方案：

打开文件时显式指定`encoding="utf-8"`

```python
# 错误示例
with open("test.txt", "r") as f:
    content = f.read()

# 正确示例
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
```
