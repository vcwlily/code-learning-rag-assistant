# code-learning-rag-assistant

# 💻 代码学习智能审查助手

基于RAG检索增强生成技术与DeepSeek代码大模型开发的编程学习辅助系统，面向编程初学者，提供代码智能分析、编程问题精准答疑能力，解决学习过程中代码看不懂、报错不会改、知识点不理解的痛点。

## 🎯 项目核心功能

1.  **代码全维度智能分析**：支持Python/Java等主流编程语言，实现语法纠错、逻辑解释、复杂度分析、优化建议四大核心能力。
2.  **基于RAG的编程智能答疑**：构建了编程学习知识库，通过两阶段检索优化，实现精准的知识点解答，严格控制大模型幻觉问题，答案可追溯、可验证。
3.  **可视化Web交互界面**：基于Streamlit开发的轻量化界面，支持代码编辑器、多轮对话、历史记录、引用来源展示，开箱即用。

## 🛠️ 技术栈选型

| 模块          | 技术选型          | 选型说明                                           |
| ------------- | ----------------- | -------------------------------------------------- |
| 后端框架      | FastAPI           | 异步高性能，自动生成接口文档，适配AI应用开发场景   |
| 前端框架      | Streamlit         | 纯Python开发，快速实现可视化交互，适配代码输入场景 |

## 🏗️ 系统架构设计

本项目采用分层架构设计，从上到下分为：

- 用户层：用户通过浏览器访问Web界面
- 前端交互层：Streamlit实现的代码输入、对话交互、结果展示界面
- 接口层：FastAPI封装的标准化RESTful接口，提供代码分析、RAG检索、智能问答能力
- 核心能力层：代码分析模块、RAG检索引擎、智能问答链
- 模型层：Embedding模型、重排序模型、DeepSeek代码大模型
- 数据存储层：ChromaDB向量数据库、编程学习知识库

## ✨ 项目核心亮点（复试重点）

1.  **针对性解决RAG检索精度问题**：设计了「Markdown标题+固定字符」的双层分块策略，保证编程文档语义完整性；采用「向量粗召回+Re-rank精排」的两阶段检索方案，经测试，问答准确率提升了30%以上。
2.  **系统性解决大模型幻觉问题**：从检索机制、Prompt强约束、模型参数优化、检索精度提升四个维度，构建了全链路的幻觉管控方案，将幻觉发生率控制在5%以内，保证答疑内容的准确性。
3.  **垂直场景深度适配**：针对编程学习场景，设计了专属的代码分析Prompt和问答规则，输出内容贴合初学者的学习需求，比通用大模型更适配编程学习场景。
4.  **完整的工程化实现**：完成了从本地开发到线上部署的全流程落地，前后端分离设计，模块解耦，具备良好的可扩展性和可维护性。
